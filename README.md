# NullSudoku datasets
Automated generation of sudoku puzzle datasets made for NullSudoku.

[![Puzzles generation](https://github.com/kinnnine/nullsudoku-datasets/actions/workflows/generate.yml/badge.svg?branch=main)](https://github.com/kinnnine/nullsudoku-datasets/actions/workflows/generate.yml)

# How?
Like [SudokuExchange.com's Puzzle Bank](https://github.com/grantm/sudoku-exchange-puzzle-bank), All puzzles are generated using [QQWing](https://github.com/stephenostermiller/qqwing) and then graded with [Sukaku Explainer](https://github.com/SudokuMonster/SukakuExplainer/), except everything is done inside Github Actions under Ubuntu. Generated per month.

This datasets is not following [SudokuExchange.com's Puzzle Bank](https://github.com/grantm/sudoku-exchange-puzzle-bank) difficulty naming and rating.
